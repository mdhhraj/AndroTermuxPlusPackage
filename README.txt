AndroTermux+ Tool Package
A lightweight utility toolkit for Termux that bundles multiple automation scripts, maintenance tools, and a central command hub.
This package provides a menu-based launcher (mymenu) that lets you manage common Termux and Android tasks quickly from one place.
Features
Central Termux Hub Menu
Termux cleanup tools
Storage maintenance scripts
System information utilities
Script logging support
Modular script structure for easy extension
Simple install and uninstall process
Project Structure
Copy code

Requirements
Before installing, make sure Termux is installed and updated.
